**Goal:** Reach PNPT exam readiness without paying £430 upfront. Pay for exam ONLY when you're confident you'll pass.

## UNDERSTANDING THE PNPT EXAM

### What You Actually Need to Pass:
1. **OSINT** - Gather intel on target network
2. **External Pentesting** - Gain initial foothold from outside
3. **Internal Network Pivoting** - Move laterally once inside
4. **Active Directory Exploitation** - Compromise Domain Controller
5. **Professional Report Writing** - Document findings professionally
6. **Live Debrief** - Present findings to assessors (15 min)

### Exam Format:
- 5 days hands-on hacking
- 2 days report writing
- 15-minute live debrief
- NO CTF flags, realistic network pentest

## CRITICAL SUCCESS FACTORS

### What Makes People Pass:

1. **Methodology Over Tools**
   - Follow systematic process
   - Document as you go
   - Don't skip enumeration

2. **Active Directory Mastery**
   - 80% of PNPT difficulty is AD
   - Know Bloodhound inside out
   - Practice privilege escalation paths

3. **Report Writing Skills**
   - Start early, practice often
   - Use templates, refine language
   - Most people underestimate this

4. **Time Management**
   - 5 days sounds long but passes quickly
   - Take breaks, sleep properly
   - Don't burn out on day 1

5. **Note-Taking During Exam**
   - Screenshot EVERYTHING
   - Document commands as you run them
   - Makes report writing 10x easier
